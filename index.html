Brendan Eich

The Creator of Javascript

Brendan Eich grew up in the northern California city of Palo Alto. He received degrees in mathematics and computer science from Santa Clara University as well as a master's degree from the University of Illinois in 1985.

His career started at Silicon Graphics working on the operating system and network code. He then moved on to MicroUnity Systems Engineering writing microkernel and DSP code.

Netscape and JavaScript

In 1995, Brendan started at Netscape Communications Corporation. There, he was tasked with developing a scripting language for the Netscape browswer which is how he went on to create what is now known as JavaScript. It was first called Mocha and later LiveScript. It took Brendan only 10 days to develop the scripting language.

Time line for the evolution of Javascript
  • 1995 JavaScript is born
  • 1996 JavaScript is handed to ECMA and officially given the name ECMAscript
  • 1998 ECMAscript 2 is released
  • 1999 ECMAscript 3 is released
  • 2005 Eich and Mozilla join ECMA
  • 2009 The CommonJS project is created
  • 2016 2% of all websites use JavaScript
